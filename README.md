# CyberShield INDIA

A web-based awareness and cyber safety platform designed to educate Indian youth about cyberbullying prevention, ethical digital communication, accessibility, and online safety through Indian Knowledge Systems principles.

## Features
- Cyberbullying Detection Simulator
- Anonymous Reporting Portal
- Awareness Quiz
- Accessibility Tools
- Live Statistics Dashboard
- Chatbot Assistance
- Cyber Safety Helplines
